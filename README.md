# Movies-Expert-System
An expert system that tries to guess the movie you're thinking of

you need to download a prolog IDE first to run this program, e.g. SWISH

1.first run the "ExpertSystem.pl" file using prolog IDE
2.type "go."
3. to answer the questions you must type ("yes." or "y.") for true questions and ("no.") for false questions
